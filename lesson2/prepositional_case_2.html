Prepositional in plural: the main pattern

Most common ending

Plural nouns usually take -–∞ћБ—Е or -—ПћБ—Е.

  • —И–Ї–ЊћБ–ї–∞ вЖТ –≤ —И–Ї–Њ–ї–∞ћБ—Е
  • –њ–∞ћБ—А–Ї вЖТ –≤ –њ–∞ћБ—А–Ї–∞—Е
  • –і–Њ–Љ вЖТ –Њ –і–Њ–Љ–∞ћБ—Е
  • —Б–µ–Љ—М—ПћБ вЖТ –≤ —Б–µ–Љ—М—ПћБ—Е
  • –Љ–ЊћБ—А–µ вЖТ –≤ –Љ–Њ—А—ПћБ—Е

Common subtype

Nouns in -–ЄћБ—П / -–ЄћБ–µ often use -–ЄћБ—П—Е.

  • –Љ—Г–Ј–µћБ–є вЖТ –≤ –Љ—Г–Ј–µћБ—П—Е
  • –∞ћБ—А–Љ–Є—П вЖТ –≤ –∞ћБ—А–Љ–Є—П—Е
  • –Ј–і–∞ћБ–љ–Є–µ вЖТ –≤ –Ј–і–∞ћБ–љ–Є—П—Е

Plural examples in context

Location

  • –Ь—ЛћБ –≤ –≥–ЊћБ—А–Њ–і–∞—Е. (WeвАЩre in cities.)
  • –Ю–љ–ЄћБ –≤ –њ–∞ћБ—А–Ї–∞—Е. (They are in parks.)
  • –Ь—ЛћБ –љ–∞ —ГћБ–ї–Є—Ж–∞—Е. (WeвАЩre on streets.)

Topic

  • –Ь—ЛћБ –≥–Њ–≤–Њ—А–ЄћБ–Љ –Њ –і—А—ГћБ–Ј—М—П—Е. (We talk about friends.)
  • –ѓ –і—ГћБ–Љ–∞—О –Њ –і–Њ–Љ–∞ћБ—Е. (I think about houses.)
  • –Ь—ЛћБ —З–Є—В–∞ћБ–µ–Љ –Њ —Б—В—А–∞–љ–∞ћБ—Е. (We read about countries.)

Adjectives in the prepositional case

Singular endings

  • Masculine/Neuter: -–ЊћБ–Љ / -–µћБ–Љ
  • Feminine: -–ЊћБ–є / -–µћБ–є

Adjective agrees with the noun: gender + number + case.

Plural endings

  • Plural: -—ЛћБ—Е / -–ЄћБ—Е

Examples (singular)

  • –≤ –±–Њ–ї—М—И–ЊћБ–Љ –і–ЊћБ–Љ–µ (in a big house)
  • –≤ –љ–ЊћБ–≤–Њ–є —И–Ї–ЊћБ–ї–µ (in a new school)
  • –Њ –Ї—А–∞—Б–ЄћБ–≤–Њ–Љ –≥–ЊћБ—А–Њ–і–µ (about a beautiful city)
  • –≤ —Б–ЄћБ–љ–µ–Љ –Љ–ЊћБ—А–µ (in the blue sea)

Examples (plural)

  • –≤ –±–Њ–ї—М—И–ЄћБ—Е –≥–Њ—А–Њ–і–∞ћБ—Е (in big cities)
  • –Њ –љ–ЊћБ–≤—Л—Е –і–Њ–Љ–∞ћБ—Е (about new houses)
  • –≤ —Б–ЄћБ–љ–Є—Е –Љ—Г–Ј–µћБ—П—Е (in blue museums)
  • –Њ —Б—В–∞ћБ—А—Л—Е –і—А—ГћБ–Ј—М—П—Е (about old friends)
